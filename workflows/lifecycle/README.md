# `workflows/lifecycle/` — Project-Level Lifecycle Workflows

These workflows are run **rarely** — at project boundaries: bootstrap, repository initialization, planning, architectural shifts.

| 01  | [Start a New Project](01-start-new-project.md)             | Once per project                   |
| 02  | [Repository Initialization](02-repository-initialization.md) | Once per repository (after 01)   |
| -   | [Planning](planning.md)                                    | Every iteration                    |
| -   | [Architecture Design](architecture-design.md)              | When designing a slice or feature |
