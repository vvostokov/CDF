# Development Guides

> Guides for contributors — humans and AI.

## What Goes Here

* Setting up a development environment.
* Running tests.
* Writing tests.
* Submitting a PR.
* Working with the cognitive layer.
* Using AI roles.

## Conventions

* Show commands, not just prose.
* Use copy-pasteable blocks.
* Explain **why**, not only **how**.

## Templates

* `setup.md`
* `running-tests.md`
* `submitting-a-pr.md`
* `working-with-cognition.md`
* `using-ai-roles.md`
